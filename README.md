1️⃣ Tela de Login
Tela inicial do sistema onde o usuário realiza autenticação utilizando matrícula/SIAPE e senha. Também oferece opções de cadastro, recuperação de senha e botão para voltar.<br><br>

2️⃣ Tela de Perguntas Frequentes (Ajuda)<br>
Seção destinada a esclarecer dúvidas comuns dos usuários, como prazos de empréstimo, multas, localização de livros e formas de contato com a biblioteca.<br><br><br>

<img width="823" height="692" alt="ajuda" src="https://github.com/user-attachments/assets/68050b39-2136-4c9d-af47-eea52deada70" /><br><br><br>

Permite o registro de novos usuários, solicitando dados pessoais, endereço, contatos e criação de senha com confirmação e critérios mínimos de segurança.<br><br>

Área onde o usuário pode visualizar e atualizar suas informações pessoais, como e-mail, telefone, endereço e complemento.<br><br>
